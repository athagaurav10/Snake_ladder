## Run Locally

Clone the project

```bash
  git clone https://github.com/athagaurav10/Snake_ladder
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pip install turtle
  pip install random 
```

Start the server

```bash
  npm run start
```
